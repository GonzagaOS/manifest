### Sync ###

```bash

# Initialize local repository
repo init -u https://github.com/Gonzaga-OS/manifest.git -b 12

# Sync
repo sync
```

### Build ###

```bash

# Set up environment
$ . build/envsetup.sh

# Choose a target
$ lunch javanese_$device-userdebug

# Build the code
$ make java -jX
```
